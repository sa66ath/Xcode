- SDK のソース
	public.sdk/
	pluginterfaces/
		SDK のソースの中身は一切変えない。
		ヘッダを利用するため、 コンパイラのインクルード・フォルダ指定に $(ProjectDir) を追加しておく。

- ソース
	fx.h
	fx.cpp
		AudioEffefctX から派生させたクラス
		信号処理本体もここに書く

	vst_param.h
		VST Plugin のパラメータ・クラス

	param.h
	param.cpp
		VST Plugin のパラメータ処理クラス
		ここで自分のパラメータを定義する
	
	bosc.h
		発振器クラス

	freq.h
	fs.h
	fs.cpp
		周波数と周期クラス
		fs.* はサンプリング周波数/周期クラス

	vstplug.def
		DLL のエントリポイントをエクスポートするため、 リンカに食わせるモジュール定義ファイル。
		本来 VSTPluginMain() を #define VST_EXPORT __declspec(dllexport) しても良いのだが
		SDK のソースの中身をいじりたく無いので、Linker のモジュール定義ファイルで EXPORT を指定する
